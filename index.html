Real-Time Weather API

About This Project

This weather app fetches real-time climate information using the OpenWeatherMap API. It supports city and Indian village names. Built using HTML, CSS, and JavaScript, it's ideal for projects, demonstrations, and real-world use with light modifications.

  • API used: OpenWeatherMap
  • Language: JavaScript
  • Frontend: HTML & CSS
  • Features: Real-time weather, icons, error handling
